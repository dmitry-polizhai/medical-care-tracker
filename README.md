# Medical Care Tracker API

REST API для хранения истории посещений врачей.

## Технологии

* Python 3.12
* FastAPI
* PostgreSQL
* SQLAlchemy
* Docker
* Swagger/OpenAPI

---

## Возможности

* Создание медицинского визита
* Получение списка визитов
* Получение визита по ID
* Частичное обновление визита
* Удаление визита

---

## Запуск проекта

### Docker

```bash
docker-compose up --build
```

### Локально

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

---

## Документация API

После запуска:

Swagger UI:

http://localhost:8000/docs

# API

## POST /visits

Создает новый медицинский визит.

### Тело запроса

```json
{
  "visit_date": "2026-05-27",
  "doctor": "Кардиолог",
  "clinic": "Евромед",
  "reason": "Боль в груди",
  "treatment": "Таблетки",
  "next_visit": "2026-06-15"
}
```

### Успешный ответ

Код: 201 Created

```json
{
  "id": 1,
  "visit_date": "2026-05-27",
  "doctor": "Кардиолог",
  "clinic": "Евромед",
  "reason": "Боль в груди",
  "treatment": "Таблетки",
  "next_visit": "2026-06-15"
}
```

### Возможные ошибки

| Код | Причина                 |
| --- | ----------------------- |
| 422 | Ошибка валидации данных |

---

## GET /visits

Возвращает список всех визитов.

### Успешный ответ

Код: 200 OK

```json
[
  {
    "id": 1,
    "doctor": "Кардиолог"
  }
]
```

---

## GET /visits/{id}

Получение визита по идентификатору.

### Успешный ответ

Код: 200 OK

### Ошибки

| Код | Причина         |
| --- | --------------- |
| 404 | Визит не найден |
| 422 | Некорректный ID |

Пример ошибки:

```json
{
  "detail": "Визит с ID 999 не найден"
}
```

---

## PATCH /visits/{id}

Частичное обновление визита.

### Пример запроса

```json
{
  "doctor": "Терапевт"
}
```

### Возможные ошибки

| Код | Причина          |
| --- | ---------------- |
| 404 | Визит не найден  |
| 422 | Ошибка валидации |

---

## DELETE /visits/{id}

Удаление визита.

### Успешный ответ

Код: 204 No Content

### Возможные ошибки

| Код | Причина         |
| --- | --------------- |
| 404 | Визит не найден |
